## 一、核心参数及其物理意义

### 1.1 b0 — 被控对象的控制增益（补偿因子）

**物理定义：**
$$b_0 \approx \frac{\dot{y}(0)}{U}$$

即：**阶跃输入下的初始加速度 / 输入幅值**。b0 代表了系统"输入→输出"的增益。

**对于四轴角速度环：**
$$b_0 \approx \frac{\text{最大角加速度}}{\text{最大控制输出}} \approx \frac{1500\ °/s^2}{300} \approx 5$$

**效果：**

| b0 | 效果 |
|----|------|
| **偏大** | 补偿不足，响应变慢，稳定但抗扰能力弱 |
| **偏小** | **补偿过度，扰动估计被放大 → 振荡/发散** |
| 合适 | 补偿刚好，又快又稳 |

> 论文结论："b0 不需要精确，±30% 误差是可以接受的。但 **b0 偏小是振荡最常见的成因**。"

---

### 1.2 wc — 控制器带宽（响应速度）

**物理定义：** 决定了闭环响应的带宽，等效于理想一阶系统的截止频率（rad/s）。

一阶 LADRC：`u0 = wc × error`（等效于 P 控制器的 Kp）

| wc | 效果 |
|----|------|
| **偏大** | 响应快，但过冲/振荡/不稳定，噪声放大 |
| **偏小** | 响应慢，稳定但跟踪滞后 |
| 合适 | 快速跟随又不震荡 |

> 论文建议："选能接受的**最小 wc**，避免噪声放大和稳定性裕度下降。"

---

### 1.3 wo — 观测器带宽（扰动估计速度）

**自动推导：** `wo = (3~5) × wc`（Gao 2003 带宽法）

| 应用场景 | wo/wc 比值 | 原因 |
|----------|-----------|------|
| 高测量噪声 | 3 | 降低噪声敏感度 |
| 通用工业控制 | 4 | 性能和鲁棒性平衡 |
| 快速跟踪 / 抗扰 | 5 | 提升估计速度 |

- ESO 增益：`β1 = 2·wo`，`β2 = wo²`（一阶）；`β1 = 3·wo`，`β2 = 3·wo²`，`β3 = wo³`（二阶）
- **wo 过大** → 扰动估计 z3 抖动剧烈、噪声放大
- **wo 过小** → 扰动估计滞后、跟踪慢

## 二、有效增益

ADRC 的瞬态响应等价于 P 控制器：

$$\text{有效 P 增益} = \frac{wc}{b_0}$$

这个值应与系统原有的 PID 的 Kp 在同一数量级，但不要求精确相等。

## 三、振荡诊断表

| 振荡类型 | 表现 | 可能原因 | 解决方法 |
|----------|------|----------|----------|
| **目标点振荡** | 稳态来回晃 | wc 太大 / **b0 太小** | 减小 wc / **增大 b0** |
| **动态过程振荡** | 打杆后晃几下才稳 | **b0 太小** / wo 太大 | **增大 b0** / 降低 wo |
| **高频震颤** | 电机刺耳、高频微抖 | wo 过大 引入噪声 | 减小 wo / 增大 b0 |
| **大幅超调** | 打杆后冲过头再回来 | wc 太大 / wo 不足 | 减小 wc / 增大 wo |
| **稳态偏差** | 有固定角度偏斜 | b0 偏大 补偿不足 | 减小 b0 |

## 四、标准调参流程

### 流程一：逐步逼近法（推荐）

```
Step 1: 粗估 b0
   b0 = 最大角加速度 / 最大控制输出
   未知时从 b0 = 1 开始，正负号与被控对象一致

Step 2: 设定 wc 初值
   取较小的 wc（如 5~10 rad/s）
   
Step 3: 设定 wo
   wo = (3~5) × wc

Step 4: 逐步增大 wc
   直到响应速度满足要求

Step 5: 出现振荡
   增大 b0（不是减小 wc）
   减小 wo（如果噪声太大）

Step 6: 微调循环
   重复 4-5 直到满意
```

### 流程二：调速优先法（更快）

```
Step 1: 固定 b0、固定较小的 wc，逐步增大 wo
        直到 z1 能较好跟踪测量值

Step 2: 逐步增大 wc，提高响应

Step 3: 超调 → 减小 wc 或增大 wo
        振荡 → 增大 b0
        
Step 4: 若始终找不到稳定组合 → 继续增大 b0 从头来
```

## 五、参数定性规律速查

| 参数 | ↑ 增大 | ↓ 减小 |
|------|--------|--------|
| **wc** | 响应加快、带宽增大、抗扰增强，但易超调/振荡/不稳定 | 响应变慢、稳定性增加 |
| **wo** | 扰动估计更快更准、噪声放大、z2 抖动大 | 估计滞后、滤波强、更稳定 |
| **b0** | **振荡减小**、稳定性增加、扰动补偿变弱 | 补偿增强、响应快、**但补偿过度会振荡** |

---

## 六、核心口诀

```
稳态振荡 → 减 wc 或 增 b0
动态振荡 → 增 b0
超调     → 增 wo 或 减 wc
高频颤振 → 减 wo 或 增 b0
```

## 七、常见问题

### 7.1 ESO 输出 NaN
- 检查 b0 是否为 0（初始化顺序错误常见）

### 7.2 控制量频繁饱和
- 加入 TD（跟踪微分器）安排过渡过程
- 或者增大 b0 降低输出幅度

### 7.3 z2 不能很好跟踪测量微分
- 属于正常现象，因为微分信号本身含噪声，不必强求

### 7.4 始终有超调
- 减小 wc + 增大 wo

## 八、无人机角速度环推荐初值

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| Ts | 0.005s | 200Hz 采样 |
| b0 | 5~10 | 约等于 max角加速度/max控制输出 |
| wc | 5~15 rad/s | 对应约 1~2.5 Hz 带宽 |
| wo | 2~3 × wc | 无人机机械系统响应慢，不必太高 |
| z2 钳位 | ±(b0 × 300 × 0.5) ≈ ±750 | 防止 ESO 模型失配时发散 |
